Phase 3: The Asymptotic Scoring Engine (The Math)

The true genius of the Sentinel-44 Python script lies in how it scores threats. New Score = Current Score + (Value ÷ 100 × (100 − Current Score))

In this math, the Value represents the raw weight of the incoming new alert, and the phrase "100 − Current Score" represents the remaining mathematical room left before hitting the absolute ceiling of 100. Because the new input is only multiplied by this remaining room, the score can climb higher and higher but will never cross 100, preventing data overflows.

To feed this formula, Sentinel-44 applies a strict weighting hierarchy based on the intelligence principle of watching actions rather than words:

- Low Weight (3–6 points): Media chatter and news articles. Talk is cheap, so headlines move the needle very little.
